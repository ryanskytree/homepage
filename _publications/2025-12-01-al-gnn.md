## Overview

AL-GNN introduces an analytic memory mechanism that enables continual graph learning without replay buffers. The framework recursively updates memory representations using recursive least squares, preserving prior knowledge while maintaining privacy.

## Research Area

*   Continual Graph Learning
*   Analytic Learning
*   Replay-Free Learning

## Keywords

Continual Learning, Graph Learning, Recursive Least Squares, Privacy-Preserving AI
</span>

## 概述

AL-GNN 引入了一种分析记忆机制，可以在没有重放缓冲区的情况下实现持续图学习。该框架使用递归最小二乘法递归更新记忆表示，在保持隐私的同时保留先验知识。

*   持续图学习

*   无重放学习
